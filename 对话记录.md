### 什么是变量？
变量就像一个盒子，用来存储数据（比如名字、数字等）。在Python中，创建变量很简单，只需要给它起个名字，然后用 `=` 号赋值。

### 如何创建变量并输出？
1. **创建变量**：比如我们想存储名字，就可以写 `name = "小明"`（这里的 `"小明"` 是字符串，需要用引号括起来）。
2. **输出变量**：使用 `print()` 函数来显示变量的内容，比如 `print(name)` 会输出 `小明`。

### 现在，我们来完成第二题：
**任务**：创建一个变量 `name`，赋值为你的名字，然后输出 "Hello, [name]!"

**示例代码**：
```python
name = "小红"  # 这里替换为你的名字
print("Hello, " + name + "!")
```

如果你运行这段代码，会输出 `Hello, 小红!`（如果你的名字是小红的话）。
